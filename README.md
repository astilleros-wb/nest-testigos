# Production deploy

  1. Clone repository
  
  2. Exec:
  ```
  npm install
  ```

  3. Install Nest CLI
  ```
  npm i -g @nestjs/cli
  ```

  4. Up Database
  ```
  docker-compose up -d
  ```
